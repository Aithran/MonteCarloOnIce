Parallellized Diffusion Monte Carlo Simulation
Example on usage:

>>> import sys, math, numpy, MLab, os, pickle
>>> from MontePython import MontePython
>>> d = MontePython(pypar)
>>> d.reset_params()

>>> d.blocks = 5
>>> d.tau_list = [[0.001,100,d.blocks],]
>>> blocks = d.blocks
