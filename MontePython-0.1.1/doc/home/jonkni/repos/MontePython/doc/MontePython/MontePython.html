Parallellized Diffusion Monte Carlo Simulation Example on usage: >>> import pypar,math Pypar (version 1.9.1) initialised MPI OK with 1 processors >>> from MontePython import MontePython >>> >>> d = MontePython(pypar) >>> # 1 particle and alpha=0.5 yields a harmonic oscillator with >>> # energy == 3/2: >>> d.params[0] = 0.5 >>> d.reset_params() >>> d.silent = True # to avoid too much noise >>>
